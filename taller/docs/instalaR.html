Introducción

Para empezar a trabajar con datos en R necesitamos preparar dos piezas que funcionan juntas: el lenguaje R y una interfaz para escribir y ejecutar nuestro código de forma cómoda. En esta guía vamos a instalar ambas y a verificar que todo quedó andando. Si en algún momento la instalación se te complica, más abajo dejo dos alternativas que te dejan trabajar sin instalar nada pesado en tu computador.

Antes de partir, conviene tener clara una distinción que suele confundir al principio:

  • R es un lenguaje y ambiente de programación, dentro del cual se pueden implementar técnicas estadísticas y de ciencia de datos. Se instala una sola vez y queda funcionando “por debajo”.
  • El IDE (RStudio, o Positron) es la interfaz que abrimos día a día para interactuar con R. Nos permite escribir código, ejecutarlo, ver gráficos y organizar nuestro trabajo.

Una analogía simple: R es el motor del auto y el IDE es el tablero desde el que conducimos (volante, palanca de cambios). Necesitamos las dos cosas, pero interactuamos casi siempre con el tablero.

Paso 1: Instalar R

R es el lenguaje base de análisis estadístico. Su gran fortaleza es que se amplía con paquetes (o librerías): pequeños complementos que agregan funciones para casi cualquier tarea, desde limpiar datos hasta hacer gráficos o modelos.

Instalamos R primero, porque tanto RStudio como Positron lo usan por detrás: sin R, el IDE no tiene con qué trabajar.

En la página de descarga elige el instalador según tu sistema operativo (Windows, macOS o Linux) y ejecútalo con las opciones que vienen por defecto.

Descargar R desde CRAN

Nota

Nota para Windows. Para algunas tareas avanzadas (compilar ciertos paquetes) es posible que además necesites Rtools. Su enlace aparece en la misma página de descarga cuando eliges Windows. Para arrancar el taller no es imprescindible, así que puedes dejarlo para más adelante.

Paso 2: Instalar RStudio

RStudio es un entorno de desarrollo integrado (IDE): reúne en una sola ventana la consola de R junto a varios paneles que nos facilitan el trabajo. Lo desarrolla Posit, funciona con R y también con Python, y hoy es la opción más usada para aprender.

Sus paneles principales son:

  • Source / script: donde escribimos y guardamos nuestros documentos con instrucciones para R.
  • Console: donde R recibe y ejecuta esas instrucciones.
  • Environment: donde vemos los objetos (datos, variables, resultados) con los que estamos trabajando.
  • Plots / Files / Help: donde exploramos los gráficos, navegamos archivos y consultamos la ayuda.

Además, RStudio nos ahorra pasos en tareas como importar datos, instalar paquetes o llevar control de versiones.

Descargar RStudio Desktop

Con R y RStudio ya instalados, abre RStudio y comprobemos que todo funciona: escribe en la consola 1 + 2 y presiona Enter. Si te devuelve 3, ¡la instalación quedó lista y estamos en marcha!

Alternativas si no puedes instalar

Si no quieres instalar nada, tu equipo es limitado, o simplemente la instalación no coopera, hay dos caminos que también funcionan perfecto para el taller.

Alternativa 1: RStudio en el navegador (Posit Cloud)

Posit Cloud te da un RStudio completo dentro del navegador web, sin instalar nada en tu computador. Solo necesitas crear una cuenta gratuita; luego, en la esquina superior derecha, elige New project → New RStudio Project y tendrás una instancia de R lista para usar, con la misma experiencia que en la versión de escritorio.

Ir a Posit Cloud

Alternativa 2: Positron (el IDE nuevo de Posit)

Positron es el IDE de nueva generación de Posit, pensado especialmente para ciencia de datos, con soporte para R y Python en un mismo lugar. Es gratuito y está construido sobre la base de VS Code (Code OSS), así que si vienes de ese mundo te resultará muy familiar.

Al igual que RStudio, Positron necesita que R (Paso 1) ya esté instalado en tu computador. Se instala como cualquier programa y se actualiza solo.

Descargar Positron

Nota

Nota para Windows. A diferencia de otras instalaciones, Positron no incluye Rtools. Si en algún momento lo necesitas, tendrás que instalarlo por separado.

Positron es una alternativa moderna a RStudio; para este taller cualquiera de los dos te sirve, así que quédate con el que te resulte más cómodo.

Con cualquiera de estas opciones vas a estar listo para el taller. Si algo no funciona como esperabas, avísame antes de la sesión y lo resolvemos con calma.
